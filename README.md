# 📚 VidyaVault – Library Management System

**VidyaVault** is a modern **Library Management System interface** designed to simplify the management of books, students, and library operations.

This project focuses on creating a **clean, modern, and responsive UI** for managing library resources efficiently. It provides an organized dashboard-style interface with smooth animations and a dark/light theme option.

---

## ✨ Features

- 📖 Book Management Interface
- 👨‍🎓 Student/User Management
- 🔍 Search functionality
- 🌙 Dark & Light theme toggle
- 🎨 Modern and clean UI design
- ⚡ Smooth animations and transitions
- 📱 Responsive layout

---

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Fonts**
- **Modern CSS variables for theming**
